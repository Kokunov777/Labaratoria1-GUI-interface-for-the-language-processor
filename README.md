## Описание проекта

Данное приложение представляет собой многофункциональный текстовый редактор с графическим интерфейсом. Редактор поддерживает стандартные операции с текстовыми файлами, редактирование текста, а также включает область вывода результатов для будущей интеграции языкового процессора.

### Ключевые возможности:

- Создание, открытие, сохранение текстовых файлов
- Поддержка различных кодировок (UTF-8)
- Операции редактирования: отмена/повтор, вырезание/копирование/вставка
- Подсветка синтаксиса Python (ключевые слова, строки, комментарии)
- Панель инструментов с часто используемыми функциями
- Область вывода с возможностью перехода к ошибкам по клику
- Изменяемый размер областей редактора и вывода
- Строка состояния с информацией о позиции курсора
- Система подсказок при сохранении изменений
- Справочная система

## Используемые технологии

- **Язык программирования:** Python 3.9+
- **GUI фреймворк:** PySide6 (Qt for Python)
- **Среда разработки:** VS Code 
- **Система сборки:** Интерпретатор Python
