## Объявление комплексного числа rust

Разработать сканер для распознавания лексем языка программирования C-подобного синтаксиса. Допустимые лексемы включают:

Ключевые слова: int, float, if, else, while, return, void, char

Идентификаторы: последовательности букв и цифр, начинающиеся с буквы или подчеркивания

Целые числа: последовательности цифр

Вещественные числа: последовательности цифр с десятичной точкой

Операторы: +, -, *, /, =, ==, !=, <, >, <=, >=

Разделители: ;, ,, (, ), {, }

Комментарии: однострочные (//) и многострочные (/* ... */)

Строковые константы: последовательности символов в кавычках "..."
